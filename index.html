CuddyIA

Cuddy es un modelo de inteligencia artificial usando la API de ChatGPT, y DALL·E.

Con este bot puedes usar ChatGPT de una manera ilimitada, también es posible generar imágenes.

Características sobre Cuddy

  • Chatear de manera fluida
  • Generación de imágenes
  • Usos ilimitados

Comandos Sobre Cuddy

    Para usar ChatGPT es el siguiente comandó:

  • C/gpt
  • /gpt-3

    • Para usar DALL·E es el siguiente comando:

  • C/image
  • /image

Metodos para añadir a Cuddy

  1. Top.gg
  2. URL invite
